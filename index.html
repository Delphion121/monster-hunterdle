Welcome to Monstle! A game where you have to guess the monster I'm thinking of! To make it easier for you, monsters from Frontier are almost entirely excluded. Additionally, sub-species are also not included - Instead, if a monster has a subspecies, it will Instead be included in the "Has Subspecies?" metric. Deviants and Variants have distanct changes that make them a bit more memorable, so I've included them. Deviants/Variants will share the same subspecies with the original monster. Example: Brute tigrex will count as a subspecies for Grimclaw and Regular Tigrex. Spinoff games and Monster Hunter Online are not considered for any metric!
